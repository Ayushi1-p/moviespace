# 🎬 Moviespace

A movie search web application that allows users to search for any movie using API integration.

---

## 🚀 Features

- 🔍 Search any movie instantly
- 📡 Fetch data using API
- ⚡ Dynamic content rendering
- 📱 Fully responsive design

---

## 🛠️ Tech Stack

- React
- HTML
- CSS
- JavaScript
- API Integration
